# Exercice 1 : Bien démarrer !

L'une des couleurs les plus utilisés pour le ton principal est le bleu, que l'on va réutiliser sur les éléments principaux des pages d'un site web. Faîtes vos premiers pas en son honneur avec Sass !

## Consignes

Dans ce dossier, vous allez trouver un autre fichier **\_bien-demarrer.scss** : c'est ici que vous pourrez directement coder !

Dans ce fichier **\_bien-demarrer.scss** :

- créez une variable `$primary-color` contenant une couleur bleue de votre choix
- appliquez cette couleur au texte de tous les paragraphes `<p>`.
